## 无损音乐下载
主要功能是下载无损音乐，数据来自 [魔声](http://moresound.tk/music/)

### Features
可以下载各大音乐平台的无损音乐。数据均来自魔声，只不过用 PyQt 包装了一下，若是下载不了可能是他们的网站崩了吧。
- QQ
- 酷我
- 虾米
- 酷狗
- 百度
- 网易

### KeyBindings
Key Binding                                | Description
-------------------------------------------|--------------------------
<kbd>Enter</kbd>                           | 搜索
<kbd>Ctrl + Enter</kbd>                    | 下载
